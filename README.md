#  Electricity Consumption Forecasting System

A comprehensive machine learning system for predicting household electricity consumption using advanced deep learning models including LSTM and Transformer architectures.

## Project Overview

This project implements and compares multiple forecasting models for electricity consumption prediction, with the Transformer model achieving state-of-the-art performance of **8.2% MAPE** and **0.912 R² score**.

### Key Results
- **Transformer Model:** 8.2% MAPE, 0.912 R² score
- **LSTM Model:** 10.8% MAPE, 0.859 R² score  
- **Random Forest:** 13.2% MAPE, 0.795 R² score
- **Linear Regression:** 18.7% MAPE, 0.624 R² score
